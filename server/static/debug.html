Debug

Send WebSocket message

Send a message to the WebSocket server. The message will be displayed in the message list above.

The following commands are available:

  • COMMAND:UPDATE_CONFIG - Forces the interface to retrieve the latest config from the server.
  • COMMAND:SEND_STATUS - Forces the interface to send its current status.
  • COMMAND:DEBUG_ON - Enables debug mode for the interface.
  • COMMAND:DEBUG_OFF - Disabled debug mode for the interface.
  • COMMAND:START_RECORDING - DEBUG:Will start a recording if the phone is off-hook. Any state changes of the phone are ignored.
  • COMMAND:STOP_RECORDING - DEBUG:Will stop the started recording.
  • COMMAND:START_PLAYBACK:{id} - DEBUG:Will play the message with the given ID.
  • COMMAND:STOP_PLAYBACK - DEBUG:Will delete the message with the given ID.
  • COMMAND:RING - Will perform the phone ring action with the configured settings.
